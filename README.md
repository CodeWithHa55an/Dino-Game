# Dino-Game

A Google Dino-inspired endless runner game built with **C++** and **Raylib**.

This project is being developed to strengthen my understanding of game development, object-oriented programming (OOP), problem-solving, and software engineering fundamentals. It is also my first game development project using Raylib.

## 🎯 Project Goals

* Learn game development fundamentals
* Practice C++ and OOP concepts
* Implement collision detection and game physics
* Improve problem-solving skills through project-based learning
* Build a portfolio-ready project

## 🚀 Features

* Player movement
* Jump mechanics
* Obstacle generation
* Collision detection
* Score tracking
* Increasing difficulty
* Game over and restart system

## 🛠️ Technologies Used

* C++
* Raylib
* Visual Studio Code
* Git & GitHub
